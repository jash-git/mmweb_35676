鑄鐵壺沖出新選擇 卡塔摩納美好結合[mmweb_35676]


資料來源: http://mmweb.tw/news/35676/


    大古鐵器24日下午邀請SCA＆CQI頂級咖啡師Josh Tien，以自身在咖啡世界專研的心得，當場以這把經典鑄鐵手沖壺沖煮咖啡、與消費者分享咖啡獨特的滋味。

    全球最大的螺旋槳製造商-般若科技總經理林允進表示：水是咖啡的靈魂，水中的礦物質含量，會直接影響萃取咖啡的風味，「好的咖啡，需要適合它的水，才能萃取出最佳風味。」

    卡塔摩納精品咖啡粉讓美好的氣氛不在被突兀的研磨聲打斷，並追求SCA金杯六大萃取理論(水質、水粉比、研磨刻度、沖煮方式、擾流、過濾材質)將定量15克精品咖啡粉，用專業的研磨技術及充氮保鮮的方式，完美呈現咖啡的美好，不但解決了粉水比及研磨刻度兩大要素的沖煮變因，更能讓美好的氛圍延續。
		


    20多年前，創辦人-張董事長在一次出差中，因緣際會結識了瓜地馬拉當地的莊園主，張董事長好奇的問:「種植咖啡的工作並不輕鬆，是什麼動力支持您堅持下去的?」老莊主緩緩地將咖啡遞給張董事長:「咖啡是生活中美好的總和，難道你不想一輩子都生活在美好裡嗎?」他輕啜一口，濃郁香醇的咖啡在唇齒間慢慢化開，這瞬間，他頓時明白了老莊主想表達的「美好」。
 
    從那天起，張董事長便對咖啡深深著迷，他義無反顧的投入咖啡事業，並不惜成本引進最新的科技與設備，也團結各方好手、綜合世界專業咖啡，組織成現在優質團隊，將他在這趟旅程中的體驗分享給大家，讓更多人瞭解咖啡的美好，豐富生活。

    在品味的道路上，我們找到一群志同道合的行家，一起為食品安全嚴格把關。不僅為生產線取得國際食品安全最嚴格的認證及清真認證，更為有機認證產品開設了獨立產線，立志要將最安心的美好傳遞到世界每個角落。


二價鐵離子的故事
    在一次偶然下，我們遇見了「大古鐵器」，他們獨特的鑄鐵技術，為咖啡的沖煮寫下了新的一頁。在測評報告中，我們又驚又喜的發現「大古鐵器手沖壺」有完整詮釋咖啡風味的能力，這表示我們因為「大古鐵器手沖壺」的出現，能為咖啡愛好者把關金杯理論的水質要素。

    原來在大古鐵器手沖壺裡的「二價鐵離子」能優化水質，與卡塔摩納精品咖啡甄選技藝結合時，不僅把關了SCA金杯理論中的水質、水粉比、研磨刻度三大要素，同時也為使用者保留了沖煮方式、擾流、過濾材質三大要素的沖煮樂趣，更為咖啡風味注入了生命與價值，讓咖啡風味更加豐富、更有溫度，也更貼近卡塔摩納想要將美好傳遞的心，是咖啡讓我們相信「原來，美好一直都在。」
